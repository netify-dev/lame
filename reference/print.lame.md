# Print method for LAME objects

Provides a concise print output for fitted LAME models

## Usage

``` r
# S3 method for class 'lame'
print(x, ...)
```

## Arguments

- x:

  an object of class "lame"

- ...:

  additional arguments (not used)

## Value

the lame object invisibly
