# Compute MCMC convergence diagnostics for multiple chains

Compute MCMC convergence diagnostics for multiple chains

## Usage

``` r
compute_mcmc_diagnostics(chain_list)
```

## Arguments

- chain_list:

  List of ame fit objects from multiple chains

## Value

List containing convergence diagnostics
