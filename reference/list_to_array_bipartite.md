# Convert bipartite list data to array format

Convert bipartite list data to array format

## Usage

``` r
list_to_array_bipartite(
  rowActorSet,
  colActorSet,
  Y_list,
  Xdyad = NULL,
  Xrow = NULL,
  Xcol = NULL
)
```

## Value

List with components `Y`, `Xdyad`, `Xrow`, `Xcol` in array format
