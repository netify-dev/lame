# Linear combinations of submatrices of an array

Computes a matrix of expected values based on an array X of predictors
and a vector beta of regression coefficients.

## Usage

``` r
Xbeta(X, beta)
```

## Arguments

- X:

  an n by n by p array

- beta:

  a p by 1 vector

## Value

An n by n matrix
