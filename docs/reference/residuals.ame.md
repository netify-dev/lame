# Extract residuals from AME model

Extract residuals from AME model

## Usage

``` r
# S3 method for class 'ame'
residuals(object, type = c("response", "pearson"), ...)
```

## Arguments

- object:

  Fitted AME model

- type:

  Type of residuals ("response" or "pearson")

- ...:

  Additional arguments

## Value

Matrix of residuals
