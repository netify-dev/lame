# Convert list to array

Convert list to array

## Usage

``` r
list_to_array(actors, Y, Xdyad, Xrow, Xcol)
```

## Arguments

- actors:

  vector of actors

- Y:

  dv in list format

- Xdyad:

  dyadic covariates in list format

- Xrow:

  sender covariates in list format

- Xcol:

  receiver covariates in list format

## Value

transforms Y, Xdyad, Xrow, and Xcol to arrays
