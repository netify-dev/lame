# Simulate Z given its expectation and covariance

Simulate Z given its expectation and covariance

## Usage

``` r
simZ(EZ, rho, s2 = 1)
```

## Arguments

- EZ:

  expected value of Z

- rho:

  dyadic correlation

- s2:

  dyadic variance

## Value

a simulated value of Z
