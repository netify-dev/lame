# Calculate memory usage of AME model components

Calculate memory usage of AME model components

## Usage

``` r
ame_memory_usage(fit, detailed = TRUE)
```

## Arguments

- fit:

  Fitted AME model

- detailed:

  Logical; show detailed breakdown (default TRUE)

## Value

Invisibly returns data.frame with memory usage
