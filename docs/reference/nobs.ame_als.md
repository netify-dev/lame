# Number of observed dyads in an ame_als fit

Mirrors
[`nobs.ame`](https://netify-dev.github.io/lame/reference/nobs.ame.md) so
the MCMC and ALS fits expose the same accessor.

## Usage

``` r
# S3 method for class 'ame_als'
nobs(object, ...)
```

## Arguments

- object:

  an `ame_als` fit.

- ...:

  ignored.

## Value

Integer: number of observed dyads.
