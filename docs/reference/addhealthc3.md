# AddHealth community 3 data

A valued sociomatrix (Y) and matrix of nodal attributes (X) for students
in community 3 of the AddHealth study.

- Y: A sociomatrix in which the value of the edge corresponds to an
  ad-hoc measure of intensity of the relation. Note that students were
  only allowed to nominate up to 5 male friends and 5 female friends.

- X: Matrix of students attributes, including sex, race (1=white,
  2=black, 3=hispanic, 4=asian, 5=mixed/other) and grade.

Originally available at `http://moreno.ss.uci.edu/data.html#adhealth`
(site no longer accessible).

## Usage

``` r
data(addhealthc3)
```

## Format
