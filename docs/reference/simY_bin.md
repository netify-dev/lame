# Simulate a network, i.e. a binary relational matrix

Simulates a network, i.e. a binary relational matrix

## Usage

``` r
simY_bin(EZ, rho)
```

## Arguments

- EZ:

  square matrix giving the expected value of the latent Z matrix

- rho:

  dyadic correlation

## Value

a square binary matrix
